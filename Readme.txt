¿Cómo ejecutar el programa?
1. Abrir la carpeta principal llamada ADA06_E16
2. Abrir la terminal
3. Escribir "javac SortingApp.java"
4. Ejecutar escribiendo "java SortingApp"
5. En ejecución, el usuario elegirá el tipo de ordenamiento, así como la columna del dataset por la cual se ordenará

Abrir javadoc en el navegador
1. Ubicarse en el directorio raiz
2. dirigirse a la carpeta de docs con el comando "cd docs"
3. Click sobre el archivo HTML llamado "index.html"
4. Click derecho en el editor
5. Seleccionar la opción "open with live server" (es necesario tener la extensión previamente)
6. Listo! Se abrirá una ventana en el navegador con la documentación.
